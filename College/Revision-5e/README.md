# Réussir son entrée en 4e

Collection de fiches de révision destinées aux élèves ayant terminé leur année de 5e.

Chaque fiche propose :

- un objectif clair ;
- trois exercices progressifs ;
- un défi plus exigeant ;
- un corrigé expliqué ;
- une synthèse des méthodes importantes.

Les ressources sont conçues pour être imprimées en noir et blanc et facilement modifiées par les enseignants.
